1. Variables & Data Types

Variables store values.

Data types are self-explanatory

  • int – whole numbers
  • float – decimals
  • str – text
  • bool – True / False
  • None – no value
x = 10  # int

2. Lists

Stores multiple ordered values in 1 variable.

numbers = [1, 2, 3]

Data can be accessed using splicing.
Be careful: Indexing starts at 0

number1[0] # returns 1
  number1[-1] # returns last value (3)

3. Dictionaries

Key–value data. Data can be accessed from keys & vice-versa.

user = {"name": "Alex"}
user["name"] # access a value using its key
user.keys()/user.values() # access all keys/values

4. Tuples

Fixed ordered values. Cannot be changed.

point = (10, 20)
point[1] # access the second value

5. Conditionals

Self-Explanatory

  • a > b # a greater than b
  • < # less than
  • >= # greater than or equal to
  • <= # less than or equal to
  • == # equal to
  • != # not equal to
  • is # same object
  • is not # not the same object
  • in # contained in
if x > 5:
    print("large")

5.5 Logic operators

These can be used to combine conditionals

  • and (&)
  • or (|)
  • not (~) # Reverses input
  • [xor] (^) # Returns True if only 1 input is True (CANNOT use word 'xor')
if x > 5 and x != 6:
    print("large & not six")

6. Loops

Repeated execution.

  • for i in range(n) # Repeats from 0 to n (end-exclusive)
  • for item in iterable # (e.g. list)
    Item can be called whatever
  • while condition # Executes while condition is True
for i in range(5):
  print(i)

Prints: 0 1 2 3 4 (each on new line)
for i in [1,2,3]:
    print(i)
number = 0
  while number < 5:
    print(number)
    number += 1

7. Functions

Reusable code blocks.

def add(a, b):
    return a + b
add(5,4) # returns 9

8. Built-in Functions

You can use these in code

  • print() #Display data
  • len() #Return length of data
  • type()
  • find() #Returns position fo value in a list
length = len(numbers)
