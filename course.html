1. Variables & Data Types

Variables store values.

Data types are self-explanatory

  • int – whole numbers
  • float – decimals
  • str – text
  • bool – True / False
  • None – no value
x = 10  # int

2. Lists

Stores multiple ordered values in 1 variable.

numbers = [1, 2, 3]

Data can be accessed using splicing.
Be careful: Indexing starts at 0

number1[0] # returns 1

3. Dictionaries

Key–value data. Data can be accessed from keys & vice-versa.

user = {"name": "Alex"}

4. Tuples

Fixed ordered values. Cannot be changed.

point = (10, 20)

5. Conditionals

Self-Explanatory

  • a > b # a greater than b
  • < # less than
  • >= # greater than or equal to
  • <= # less than or equal to
  • == # equal to
  • != # not equal to
  • is # same object
  • is not # not the same object
  • in # contained in
if x > 5:
    print("large")

6. Loops

Repeated execution.

  • for i in range(n)
  • for item in iterable
  • while condition
for i in range(5):
    print(i)

7. Functions

Reusable code blocks.

def add(a, b):
    return a + b
add(5,4) # returns 9

8. Built-in Functions

You can use these in code

  • print() #Display data
  • len() #Return length of data
  • type()
  • find() #Returns position fo value in a list
length = len(numbers)

9. Error Handling

Handle failure.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("error") # Attempts to execute "try" block, if error is raised (in this case ZeroDivision): execute "except" block.

10. Imports

Additional function from external modules.

import math
print(math.sqrt(9))
