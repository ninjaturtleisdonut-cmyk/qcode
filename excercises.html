Python Exercises

Start from basic. Try to solve from memory.
I would recommend coding in: Visual Studio

Basic (1–10)

  1. Store a whole number in a variable. Display its value and identify what type of data it is.
  2. Store three numbers in a single variable. Display the second number.
  3. Store a person’s name using a key–value structure, then display the name using its key.
  4. Store two coordinates as a fixed pair of values and display both values.
  5. Store a number. If it is greater than 5, display the word “large”.
  6. Use a loop to display the numbers from 0 up to (but not including) 5.
  7. Create a reusable block of code that takes two numbers and returns their sum.
  8. Store several values in a collection and display how many items it contains using a built-in tool.
  9. Write code that attempts a calculation that could fail, and display a message instead of crashing.
  10. Import a built-in module and use it to perform a mathematical calculation.

Intermediate (11–20)

  1. Write a program that displays all numbers between 1 and 10 that are greater than 5.
  2. Store several numbers in a collection and display only the even values.
  3. Create a reusable block of code that compares two numbers and returns the larger one.
  4. Store data using key–value pairs and use a loop to display each key.
  5. Count how many values are in a collection without using any built-in counting tools.
  6. Write a loop that continues running until a number reaches 10.
  7. Create a program that checks whether a specific value exists inside a collection.
  8. Write a program that finds the smallest number from a collection of ten values.
  9. Safely divide two numbers so the program does not crash if division fails.
  10. Calculate and display the average of several numbers stored in a collection.
